Clip Studio ES

Este repositorio es principalmente una app de escritorio (Python/Tkinter + scripts de video). El deploy en Vercel queda configurado para evitar errores de react-scripts build.

  • GUI local: python scripts/clip_studio_gui.py
  • TikTok Direct Post API: python scripts/tiktok_direct_post_api.py -h
  • Variables: TIKTOK_CLIENT_KEY, TIKTOK_CLIENT_SECRET, TIKTOK_REDIRECT_URI

Repositorio: github.com/SrSainz/tiktok
