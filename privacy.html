Clip Studio ES

Política de privacidad

Última actualización: 4 de abril de 2026

Clip Studio ES procesa enlaces de vídeo, archivos temporales de trabajo y metadatos mínimos necesarios para generar clips, revisarlos y prepararlos para publicación.

Datos que puede tratar la aplicación

  • URLs de vídeos proporcionadas por el usuario.
  • Archivos de vídeo generados y recursos temporales de edición.
  • Metadatos técnicos de publicación, como título, caption y hashtags sugeridos.
  • Tokens de acceso de integraciones externas, cuando el usuario conecta servicios como TikTok.

Finalidad

Los datos se utilizan exclusivamente para descubrir contenido, generar clips, enviarlos a revisión en Telegram y, si el usuario lo aprueba, publicarlos en TikTok.

Servicios de terceros

La aplicación puede interactuar con YouTube, Telegram y TikTok. Cada integración se utiliza solo para la funcionalidad solicitada por el usuario y queda sujeta también a las políticas de privacidad de esos proveedores.

Conservación

Los archivos y tokens se conservan en la infraestructura controlada por el usuario o por el responsable del proyecto mientras sean necesarios para el funcionamiento de la herramienta.

Seguridad

Se aplican medidas razonables para proteger tokens y archivos de trabajo, pero ningún sistema conectado a internet puede garantizar seguridad absoluta.
